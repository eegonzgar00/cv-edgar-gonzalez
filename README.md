# Edgar González — CV / Portfolio

Personal CV and portfolio site built as a single HTML/CSS/JS file, deployed via GitHub Pages.

**Live site:** https://eegonzgar00.github.io/cv-edgar-gonzalez/

## About

Modern black and green design showcasing professional experience, skills, education, and contact info for Edgar González — Senior Software Engineer with 8+ years in Java, JavaScript, Liferay, Docker, Kubernetes, and Generative AI.

## Features

- Single-file (HTML + CSS + JS, no build step, no dependencies)
- Bilingual ES/EN toggle
- Scroll-reveal animations with IntersectionObserver
- Responsive design (mobile, tablet, desktop)
- Bento grid layout for the About section
- Vertical timeline for work experience
- Animated skill bars and counters
- Bottom floating nav with active section detection

## Tech stack

| Layer    | Details                                      |
|----------|----------------------------------------------|
| Markup   | HTML5 semantic elements                      |
| Styles   | CSS custom properties, Grid, Flexbox         |
| Fonts    | Syne (headings) + DM Sans (body) via Google Fonts |
| Scripts  | Vanilla JS — no frameworks                   |
| Hosting  | GitHub Pages (branch: `master`)              |

## Structure

```
cv-edgar-gonzalez/

```

## Deploy
